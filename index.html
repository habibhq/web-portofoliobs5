About Me

My Projects

SIKOJAR (Sistem Informasi Kos Fajar) adalah proyek UI/UX yang saya kerjakan untuk mata kuliah Interaksi Manusia dan Komputer. Proyek ini berupa desain antarmuka berbasis Figma untuk sistem pengelolaan kos, tanpa implementasi koding. Saya berkontribusi dalam pembuatan mockup, ERD, serta analisis sistem lama dan rancangan sistem baru. Fokus utama proyek ini adalah kemudahan penggunaan bagi admin dan penyewa kos

Ini adalah web portofolio pertama saya yang dibuat menggunakan Bootstrap 3, sebuah framework CSS populer untuk membangun website responsif dengan cepat dan mudah. Web ini menampilkan profil singkat, daftar proyek, dan kontak saya dalam tata letak yang rapi dan mobile-friendly. Dengan Bootstrap 3, saya belajar mengatur grid, komponen seperti navbar, tombol, dan membuat desain yang konsisten tanpa perlu menulis banyak CSS dari awal. Proyek ini menjadi dasar saya dalam mengembangkan kemampuan front-end dan desain web.

Di mata kuliah Basis Data, saya mengerjakan tugas menganalisis data penjualan PUMA menggunakan Looker Studio. Tugasnya meliputi mengolah data, membuat dashboard dengan berbagai grafik, menganalisis performa penjualan berdasarkan metode dan wilayah, serta menyusun laporan dan presentasi hasil analisis untuk membantu pengambilan keputusan bisnis.
